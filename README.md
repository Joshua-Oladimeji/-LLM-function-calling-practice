 LLM Function Calling Practice
 
This repository contains structured exercises focused on JSON-based function calling, schema validation, and decision-making logic used in LLM systems.

Purpose
To simulate real-world AI tool-use systems where:

Inputs are structured as JSON
Functions are invoked via strict schemas
Outputs must follow deterministic formatting rules
What This Covers
JSON structure and validation
Function calling patterns
Decision logic (tool vs natural language)
Nested schema handling
Data type enforcement
Technologies Used
Python
JSON
VS Code
Key Principle

Precision in structure is more important than complexity of logic.

This repo is a record of progressive learning from basic to advanced structured output handling.
